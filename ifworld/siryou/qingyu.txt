# **青嶼共和国・IF世界総合資料（2025年時点）**

## **1. 地理・環境**

* **位置**：東シナ海、日本（九州）と中国（福建省）の中間。新火山活動で19世紀末に隆起。
* **面積**：約2,500 km²
* **地形**：中心部は山脈、沿岸は砂浜と港湾地帯。河川がいくつか流れる。
* **気候**：温暖湿潤、台風年1〜2回直撃。冬は温暖、夏は湿気が高い。
* **自然資源**：漁業資源豊富、海底ガス田・希少鉱石が存在。

---

## **2. 言語・文化**

* **公用語**：青嶼語

  * 広東語・周辺中国方言・日本語を基礎にした混成言語
  * 文法は広東語に準じ、語彙に日本語の影響を多数反映
* **宗教**：仏教・道教・キリスト教（折衷型）、民間信仰あり
* **建築**：和洋中折衷、港湾都市にはオスマン風建築も点在
* **伝統行事**：

  * 青嶼港祭り：年1回、伝統舞踊・音楽・海上パレード
  * 火山祭り：火山神を祀る伝統行事
* **衣食**：中華料理に日本的調理法を融合。港町の魚介料理が名産。

---

## **3. 政治・外交**

* **政府**：議会制共和制

  * 大統領：国家元首兼行政府トップ
  * 二院制議会：上院（地域代表）、下院（人口比例）
* **外交方針**：

  * 中立主義を掲げ、近隣列強と均衡を維持
  * 日本・中国・ロシアとの貿易を重視
  * 国際連盟・国連に加盟（IF世界で国際機構存在）
* **軍事**：

  * 小規模沿岸防衛海軍、民兵制度
  * 主力は港湾防衛と海洋警備
  * 東シナ海中立海域維持を優先

---

## **4. 経済**

* **主要産業**：

  * 港湾貿易・中継貿易（東アジア・東南アジア間）
  * 漁業・海洋資源開発（魚類、海底ガス）
  * 観光業（温暖な気候・文化的観光）
* **通貨**：青嶼ドル（QYD）、金本位制採用（1920年代以降）
* **主要貿易相手国**：日本、中華民国、ロシア帝国、列強諸国
* **特色**：中立港としての金融・貿易ハブ機能

## **5. 世界史IFタイムライン**

| 年代              | 世界情勢                                                                 |
| --------------- | -------------------------------------------------------------------- |
| **1890s**       | 火山隆起で島出現。日本・中国漁民が同時に発見。                                              |
| **1904-1905**   | 日露戦争：日本敗北。東郷平八郎不在の影響。ロシアが優勢。                                         |
| **1905-1910**   | 青嶼独立宣言。列強の貿易拠点として発展開始。                                               |
| **1910s**       | ロシア帝国が朝鮮・モンゴル・内モンゴルを支配。日本は敗北影響で海外進出弱まる。                              |
| **1914-1918**   | 第一次世界大戦：中央同盟国勝利。ドイツはメーメル・ロシア領ポーランドのみ獲得。損害甚大で他領土要求は失敗。                |
| **1914**        | オスマン帝国存続（現実1914年の国境維持）。                                              |
| **1920s**       | 青嶼経済安定。中立港として列強商人が滞在。観光産業も発展。                                        |
| **1930s**       | 日本は日露敗北の影響で近代化が遅れる。中国は中華民国として存続。世界的には列強均衡状態。                         |
| **1940s**       | 第二次世界大戦：明確な勝者なし（痛み分け）。青嶼は中立維持。                                       |
| **1950s-1970s** | 冷戦期：青嶼は中立外交で繁栄。港湾・観光経済が安定。                                           |
| **1980s-2000s** | 東アジア経済成長期：青嶼は中継貿易で繁栄、観光業も拡大。                                         |
| **2025年**       | 世界は二極化均衡状態。青嶼共和国は中立を維持。近隣諸国：中華民国存続、日本は近代化済み、ロシア帝国健在（旧領域維持）、オスマン帝国存続。 |

## **6. 地域情勢・外交バランス（2025年）**

* **日本**：日露敗北の影響で海外進出は限定的、国内近代化は完成
* **中華民国**：清朝滅亡後、共和制を維持、青嶼と経済関係良好
* **ロシア帝国**：朝鮮・モンゴル・内モンゴルを支配。東アジアの軍事的優位
* **ドイツ**：中央同盟国勝利国だが損害大きく、東欧のみ領土拡大
* **オスマン帝国**：第一次世界大戦後も存続、バルカン半島と中東に強い影響
* **アメリカ**：メキシコとの戦争に敗北、対外拡張は制限
* **青嶼共和国**：中立、経済中心地として繁栄、観光・港湾貿易のハブ

---

## **7. 特色・チャームポイント**

* **歴史的ユニーク性**：

  * 日露戦争日本敗北の影響で「中立の楽園」形成
  * 列強の影響を受けつつ独自文化が発展
* **文化的魅力**：

  * 青嶼語：東西文化融合の言語
  * 港町建築：和洋中折衷
  * 年中行事：港祭り・火山祭り
* **経済・戦略的魅力**：

  * 東アジアの中継貿易拠点
  * 海洋資源（魚類・天然ガス）
  * 国際中立港として観光・金融ハブ
